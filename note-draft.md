# ニコニコの「いいね・マイリスト」をXへ自動投稿する拡張機能「NicoRepo2」

ニコニコ動画で行った「いいね！」や公開マイリストへの登録を検知し、Xへ自動投稿するChrome / Edge向け拡張機能です。

旧ニコレポのTwitter連携に近い使用感を目指しています。X APIや外部サーバーは使わず、PCでログイン済みのX投稿画面を一時的に操作します。

## 主な機能

- ニコニコ動画の「いいね！」をXへ自動投稿
- 公開マイリストへの登録をXへ自動投稿
- PC上の操作だけでなく、スマホ側で行った操作も定期同期
- 同じ動画・同じ行動の二重投稿を防止
- デバイス規制動画を判定し、必要な場合だけURLを`nicovideo.gay`へ変更
- 成功・失敗・診断ログを設定画面で確認
- 1日の自動投稿上限は初期25件。設定画面から変更可能

投稿例:

```text
【いいね！】動画タイトル
https://www.nicovideo.jp/watch/sm12345678
#sm12345678 #ニコニコ動画
```

## 動作の流れ

```text
ニコニコで操作
  ├─ いいね！
  └─ 公開マイリストへ登録
          ↓
NicoRepo2が登録成功を確認
          ↓
重複・公開状態・日次上限を確認
          ↓
X投稿画面を小さな一時ウィンドウで開く
          ↓
本文入力 → 自動投稿 → ウィンドウを閉じる
```

スマホ連携の場合は、PC側の拡張機能がログイン中のニコニコアカウントを定期確認し、新しく増えた操作だけを同じ投稿処理へ送ります。

## 導入方法

NicoRepo2はChrome Web Storeではなく、開発者モードで導入します。

1. 配布されたNicoRepo2のZIPファイルを展開します。
2. Chromeで`chrome://extensions/`を開きます。
3. Edgeの場合は`edge://extensions/`を開きます。
4. 画面右上の「デベロッパーモード」をONにします。
5. 「パッケージ化されていない拡張機能を読み込む」を押します。
6. 展開したフォルダ内の`NicoAutoReport`フォルダを選択します。

更新版へ入れ替えた場合は、拡張機能一覧にある更新ボタンを押すか、NicoRepo2の再読み込みボタンを押してください。

## 最初の設定

1. PCのChrome / Edgeでニコニコ動画へログインします。
2. 同じブラウザでXへログインします。
3. NicoRepo2の詳細画面から「拡張機能のオプション」を開きます。
4. 「いいね投稿」「マイリスト投稿」がONになっていることを確認します。
5. スマホ側の操作も回収する場合は「スマホ連携」をONにします。
6. 設定を保存します。

スマホ連携の初回確認では、現在の履歴を基準として保存するだけで投稿しません。それ以降に増えた操作が自動投稿の対象になります。

## 公開マイリストだけが対象

非公開マイリストの内容をXへ投稿するとプライバシー上の問題になるため、NicoRepo2は公開状態を確認できたマイリストだけを対象にします。

非公開マイリスト、または公開状態を安全に確認できないマイリストは投稿しません。非公開から公開へ切り替えた場合も、以前から入っていた動画をまとめて投稿せず、切り替え後の新規登録だけを検知します。

## 1日の投稿上限

初期設定では、Xへの自動投稿は1日25件までです。

上限へ到達した後のイベントは削除されず、翌日までキューに保留されます。PCのローカル日付が変わると処理を再開します。上限はオプション画面から変更できますが、Xのアカウント制限リスクを理解した上で調整してください。

同じXアカウントを複数PCや複数ブラウザプロファイルで使う場合、投稿数は端末間で合算されません。複数環境では各端末の上限を低めにしてください。

## デバイス規制動画のURL

ニコニコのデバイス規制に該当する動画は、通常のURLではX上にサムネイルが表示されないことがあります。

NicoRepo2は投稿直前にニコニコ本体へ確認を行い、`HARMFUL_VIDEO`と判定された場合だけ投稿URLを`nicovideo.gay`へ変更します。規制されていない動画は通常の`nicovideo.jp`のまま投稿します。

## 注意事項

- X APIは使用していません。
- 外部サーバーへログイン情報や履歴を送信しません。
- Cookie、パスワード、認証情報は保存しません。
- データはブラウザ内の`chrome.storage.local`に保存されます。
- PC側のChrome / Edgeを終了している間は、スマホ操作の定期確認は行われません。
- Xやニコニコの画面・API仕様変更により動作しなくなる可能性があります。
- XのCAPTCHA、追加認証、ロック、投稿制限を検知した場合、突破せず投稿を中止します。
- 新規Xアカウントや短時間の大量投稿では、ラベル・ロック・表示制限などの対象になる可能性があります。

完全自動投稿を行う拡張機能です。最初は低い投稿頻度で試し、設定画面のログを確認しながら利用してください。
